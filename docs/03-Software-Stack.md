# 03 – Software Stack

SOC_Replay combines virtualization, containerization, orchestration, IaC, monitoring, logging, and automation tooling to deliver a flexible, secure lab environment.  

## Components & Tools

- **Virtualization / Containerization** — hypervisors for VMs, container engines for lightweight services  
- **Configuration Management / IaC** — declarative tools (e.g., SaltStack) to define and manage infrastructure  
- **Automation & Orchestration** — scripts and AI-driven modules for deployment, scaling, network reconfiguration, threat response  
- **Monitoring & Logging** — centralized logging (e.g., ELK stack), deep packet inspection, resource metrics  
- **Dashboard & Visualization** — real-time dashboards (Grafana) for system status, security events, network activity  
- **Security Tools** — honeypots, IDS/IPS systems, tar-pit services, traffic isolation, containment
